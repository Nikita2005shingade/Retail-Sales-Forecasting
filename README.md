# Retail Sales Forecasting

## Project Overview

This project predicts future retail sales using Machine Learning and Time Series Analysis on the Walmart Sales Dataset.

### Objectives

* Forecast future sales
* Analyze sales trends and seasonality
* Identify factors affecting sales
* Support inventory planning and business decisions

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* XGBoost
* Prophet
* Streamlit

### Project Workflow

1. Data Collection and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Time Series Analysis
5. XGBoost Model Development
6. Prophet Forecasting
7. Model Evaluation
8. Streamlit Dashboard Deployment

### Results

* Generated sales trend analysis
* Identified seasonal patterns
* Built forecasting model using XGBoost
* Created interactive dashboard using Streamlit

### Dataset

Walmart Weekly Sales Dataset from Kaggle
